## Разделы системы

### 📔 Дневник
**Функционал:**
- Добавление записей с датой, местом, названием и текстом
- Автоматическое определение геолокации
- Свернутый/развернутый вид записей
- Удаление записей
- **Мобильная кнопка:** "+ Добавить запись"

**Хранение:** localStorage (`diaryNotes`)

### ⭐ Вишлист
**Функционал:**
- Добавление желаемых вещей с названием, ссылкой и описанием
- Автоматическое определение изображения по ссылке
- Загрузка своих фотографий
- Отображение превью изображений
- Удаление items
- **Мобильная кнопка:** "+ Добавить желание"

**Хранение:** localStorage (`wishlistItems`)

### 🍳 Рецепты
**Функционал:**
- Добавление рецептов с ингредиентами и инструкцией
- Указание времени приготовления и количества порций
- Развернутый/свернутый вид
- Форматированный список ингредиентов
- Удаление рецептов
- **Мобильная кнопка:** "+ Добавить рецепт"

**Хранение:** localStorage (`recipes`)

### ✅ Список дел
**Функционал:**
- Добавление задач с приоритетом (высокий/средний/низкий)
- Установка срока выполнения
- Отметка выполнения (checkbox)
- Цветовая индикация приоритета
- Сортировка по статусу и приоритету
- Удаление задач
- **Мобильная кнопка:** "+ Добавить задачу"

**Хранение:** localStorage (`todos`)

### 🎬 Медиатека
**Функционал:**
- Добавление медиа: фильмы, сериалы, книги, игры, музыка
- Оценка от 1 до 5 звезд
- Отметка "просмотрено"
- Группировка по типам медиа
- Удаление элементов
- **Мобильная кнопка:** "+ Добавить медиа"

**Хранение:** localStorage (`mediatekaItems`)

## Общие возможности

- **Авторизация:** два статичных пользователя с обязательным входом
- **Адаптивный дизайн:** идеально работает на мобильных устройствах
- **Glassmorphism стиль:** современный дизайн с размытием
- **LocalStorage:** все данные сохраняются локально
- **Анимации:** плавные переходы и эффекты
- **PWA:** установка как приложение на телефон

## Структура файлов

### Основные файлы:
- `index.html` - главная страница с PWA поддержкой
- `login.html` - страница входа (улучшенная)
- `style.css` - общие стили с мобильной адаптацией
- `auth.js` - система аутентификации

### JavaScript модули:
- `notes.js` - управление дневником
- `wishlist.js` - управление вишлистом
- `recipes.js` - управление рецептами
- `todolist.js` - управление списком дел
- `mediateka.js` - управление медиатекой
- `geolocation.js` - определение местоположения
- `quotes.js` - цитаты дня
- `weather.js` - погода
- `glass.js` - эффекты стекла

### PWA файлы:
- `manifest.json` - манифест приложения
- `sw.js` - service worker для офлайн работы
- `browserconfig.xml` - конфигурация для Windows

### Страницы разделов:
- `pages/diary.html` - дневник с мобильной кнопкой
- `pages/wishlist.html` - вишлист с мобильной кнопкой
- `pages/recipes.html` - рецепты с мобильной кнопкой
- `pages/todolist.html` - список дел с мобильной кнопкой
- `pages/mediateka.html` - медиатека с мобильной кнопкой

- **Модульная архитектура:** каждый раздел имеет свой JS-файл
- **Единый стиль:** общие CSS классы для всех разделов

## Использование

### На компьютере:
1. Войдите в систему используя один из логинов
2. Используйте кнопку "+ Запись" на главной странице для добавления контента
3. Навигируйте по разделам с помощью меню

### На телефоне:
1. Откройте сайт в браузере
2. **Обязательно введите логин и пароль** (теперь требуется на всех устройствах)
3. Используйте удобные кнопки добавления в каждом разделе
4. Навигируйте с помощью нижнего меню
5. **Установите как приложение** через меню браузера

### Установка PWA:
1. Откройте сайт в Chrome/Safari
2. Нажмите "Добавить на главный экран"
3. Используйте как обычное приложение
