## 🚀 Быстрый старт

1. Откройте `index.html`
2. Войдите под одним из пользователей
3. Используйте "+ Запись" для добавления контента
4. Навигируйте по разделам через меню

Все данные автоматически сохраняются и доступны при следующем входе!
